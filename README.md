## SLM Chamber Visualizer

Веб-приложение для визуального моделирования внутренней камеры построения SLM-принтера.

### Возможности

- интерактивная 3D-сцена на Three.js;
- настройка сетки, камеры и дисторсии сканатора;
- моделирование нанесения крестиков на стол;
- экспорт параметров сцены, камеры и сканатора;
- HTTP API и Swagger-документация;
- сохранение снимков с виртуальной камеры.

### Запуск

```powershell
python api_server.py
```

После запуска:

- приложение: `http://127.0.0.1:8000`
- документация API: `http://127.0.0.1:8000/docs`
